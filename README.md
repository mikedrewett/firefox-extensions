# firefox-extensions

## steam-cards

### Description
Displays trading card information and price data on your Steam wishlist page.

### Installation
1. Download the latest version of the extension from the [releases](steam-cards/releases) page.
2. Within Firefox open the extensions menu.
3. Select the gear icon and then `Install Add-on from file`.
4. Select the downloaded .xpi file.

### Updates
Updates are handled automatically by Firefox. The [updates.json](steam-cards/updates.json) file contains information about available updates.
